# 如何开通 OpenAI Sora?

# 什么是 Sora

Sora 是 OpenAI 于 2024 年 2 月 17 日凌晨发布的新的文生视频大模型。

OpenAI 在官网展示了众多使用 Sora 生成的视频，从现实场景到动画片甚至还有小众的艺术风格全都具备。并且这些视频都可以仅仅通过一段文本描述即可生成，目前最长能生成 1 分钟。
与之前 Pika、Runaway、SVD-XT 等主流视频生成产品不同，从镜头丝滑程度、画面连贯性、到物体间的碰撞交互、光影表现，各方面都达到了几乎降维打击的提升。

Sora 官网

OpenAI 展示文生视频工具 Sora，效果惊艳，48 段视频 + 提示词展示

# Sora 模型能做什么？

除了招牌的文生视频功能（text-to-video），Sora 模型还能做到以下事情：

  • 从已有图像生成视频
  • 对已有视频进行前后扩展
  • 用文本描述来修改已有视频
  • 为多个视频生成无缝衔接的段落

上面的每一项功能都让人感到无限的可能性，而 OpenAI 也在其技术报告中展示了生成过程：

Sora 官方技术报告

拆解 OpenAI 技术报告：Sora 是怎么生成视频的？

当然，上面技术报告中也展示了一些生成失败的案例，表明了模型还有一些局限性，主要在于准确模拟复杂的物理世界方面还有欠缺。

综上所述，Sora 模型掀起又一轮 AIGC 热潮，一旦向公众开放，势必影响整个视频行业的发展，是否准备好迎接了呢？

# 如何开通

截止目前，Sora 模型没有针对大众开放。

在演示效果已如此惊艳的情况下，对于 ToC 产品来说还有计算力、成本、时间、合规等方面需要时间进行准备，也合情合理。
但像之前 GPT-4 内测一样，可以预见到开通了 ChatGPT Plus 的用户会有机会在未来第一时间体验 Sora 模型的功能。
